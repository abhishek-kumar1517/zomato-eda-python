# 🍽️ Zomato Data Analysis Dashboard

## 📁 Project Title
Zomato Data Analysis (EDA): Restaurant Ratings, Pricing & Cuisine Insights

 ---

## 🧩 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Zomato restaurant dataset to uncover insights related to restaurant ratings, pricing trends, popular cuisines, and location-based distribution.
The analysis helps understand customer preferences and market patterns in the food and restaurant industry.

----

## ⚙️ Tech Stack

-  Python – Core programming language for analysis
-  Pandas & NumPy – Data cleaning, transformation, and feature handling
-  Matplotlib & Seaborn – Data visualization and pattern discovery
-  Jupyter Notebook – Interactive analysis and documentation

## 🎯 Project Highlights

### 📊 Key Metrics (KPIs)
-  Total Restaurants Analyzed: Based on dataset size
-  Average Rating Range: Majority of restaurants fall in mid-rating range
-  Popular Price Segment: Affordable to mid-range restaurants dominate
-  Top Cuisines: North Indian, Chinese, Fast Food (dataset-dependent)

---

### 📈 Visual Insights

-  Rating Distribution: Histogram showing spread of restaurant ratings
-  Cost vs Rating Analysis: Relationship between pricing and customer ratings
-  Top Cuisines: Bar chart highlighting most common cuisines
-  City-wise Distribution: Restaurants concentration across locations
-  Online Delivery Impact: Comparison of ratings with delivery availability

---

### 🧾 Dataset Overview

The dataset used in this project is not uploaded to the repository due to GitHub file size limitations (file size exceeds 100 MB).

**Dataset Fields:**
- Restaurant Name
- Location / City
- Cuisines
- Average Cost for Two
- Rating
- Votes
- Online Delivery
- Table Booking

Total Records: Dataset-dependent
